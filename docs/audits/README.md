# Audit Reports

This directory contains audit reports generated by PDD audit commands:

- `/pdd-audit-devops` - DevOps and infrastructure audits
- `/pdd-audit-security` - Security vulnerability audits
- `/pdd-audit-production` - Production readiness audits
- `/pdd-audit-all` - Combined full audits

## Report Naming

Reports follow this naming convention:

```
PRD-XXX_[Type]_Audit_YYYY-MM-DD.md
```

Examples:

- `PRD-011_DevOps_Audit_2026-01-29.md`
- `PRD-011_Security_Audit_2026-01-29.md`
- `PRD-011_Production_Audit_2026-01-29.md`
- `PRD-011_Full_Audit_2026-01-29.md`
